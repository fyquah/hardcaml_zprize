Module Field_ops_lib.Modulo_adder_pipe

Computes (a + b mod P) over multiple pipeline stages.

This module requires a < P and b < P. Functionally, it implements the following over multiple pipeline stages.

mux2 ((a + b) >= P)
  ((a + b) - P)
  (a + b)
