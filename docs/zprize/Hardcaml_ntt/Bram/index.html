Module Hardcaml_ntt.Bram

Double buffered (Ultra)ram blocks which provide 2 read and 2 write ports.

Internally we instantiate 2 seperate rams to provide the require port density. The write ports access one ram, while the read ports access the other. The flip signal swaps which rams are read and written.
